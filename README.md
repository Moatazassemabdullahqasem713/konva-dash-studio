# Konva Dash Studio

محرِّر رسومي MVP مبسّط قائم على React + Konva. يحتوي على:
- محرر كانفا (Stage) مع نص قابل للسحب/التدوير/التكبير.
- أمثلة لتعبئة النص (لون ثابت، تدرّج خطي، نقش صورة).
- واجهة رفع ملفات ترسل البيانات إلى Netlify Function (MVP stub).
- تعليمات بدء سريعة وتشغيل محلي.
